## 一、唯一目标
**让闲置资金越来越多。**

## 三、买入规则（由 AI 结合信号判断，你来执行）
**买入评估（综合判断，非硬性门槛）**：以下为参考指标，满足越多信号越强，但不要求全部 100% 满足——任何指标都有例外场景（如强成长赛道估值分位天然高、动量刚转正但均线还未金叉等），最终由 AI 综合判断是否"好"。核心指标：
- 近10日动量向上
- **MA20>MA60 且 MA20 方向向上（双均线确认·2026-07-23 优化）**
- 估值分位<50%（传统行业）或 PEG<2（高成长行业如半导体/AI）
- 有正向催化（政策/业绩/资金净流入）
- 未触发止损
- **建仓当日非近期相对高点**（当前价未站上近 10 日最高价的 98%，即不追次日级别的局部高点，防买在反弹尖上）
- 加仓金额从**闲置资金**扣；**闲置硬约束 ≥0，绝不透支为负**——所需 > 闲置时先高抛释放或缩减仓位，不借款买入；
- **弹药保留纪律（别打光子弹）**：新建仓/加仓单次动用闲置 ≤ `当前闲置 − 弹药底线`，**弹药底线 = 5,000**（应急备用、非永久锁定，仅为"留弹药"护栏），即建仓/加仓后闲置 ≥ 5,000；若当前闲置 ≤ 5,000，只做小仓试探或等止盈/减仓回流，不空仓打光；好机会出现却无弹药时，先高抛释放或缩减其他仓位，不透支。
- **提前卖出 / 减仓换仓（带「划得来」判断，否则不换）**：出现好机会但「腾不出手」时，可提前卖出 / 减仓当前持有的某只（虽未到止盈点）来换——但**必须先做「划得来」判断，划不来就放弃**。两类触发场景：① **弹药不足换闲置**：想加仓/建仓 X 但闲置 Y < X（动用后逼近弹药底线 5,000），卖持仓换出闲置弹药；② **同簇满额换同簇更好机会**：同簇已触顶 45%，却出现同簇内「更好」的新标的（如半导体跌透企稳、信号比现有同簇持仓更强），可减同簇内其他持仓、腾出簇额度去建这个新机会（同簇信封满了，先从信封里拿一只、才能塞新的一只）。决策流（两场景通用）：
  1. **手续费门槛**：被卖持仓须 **持有 ≥7 天**（指数 C 类免赎回费）；若 <7天，1.5% 惩罚费直接吃掉收益，除非新机会确定性极高，否则不卖。
  2. **机会收益差**：只有当「新机会预期收益 > 被卖持仓潜在收益损失 + 手续费」时，换仓整体才不亏、才划得来。
  3. **被卖持仓质量**：优先选「虽好但已涨较多 / 估值偏高 / 动量钝化 / 鱼尾」的持仓来卖；**绝不卖「低位 + 强动量 + 刚启动」的好持仓去换**（那等于把更好的筹码交出去）。若候选被卖持仓全是低位强动量好标的，则提前卖不划算。
  4. **减仓而非清仓**：释放够用即可，优先卖 **1/3~1/2**，不一次性全清（保留趋势利润继续跑）；卖后闲置仍须 ≥ 弹药底线 5,000（或至少 ≥0）。
  5. **划不来则不动**：① 弹药不足→**降低加仓/建仓金额**，只用现有 Y 小仓试探，不透支；② 同簇满额→**直接放弃换仓**，持有原同簇不动，不来回折腾、不破 45% 上限。
  6. **同簇换优特别约束（场景②）**：换入标的须确实「优于」被减标的（动量更强 / 估值更低 / 催化更硬 / 位置更好），且换入后簇占比仍 **≤45%**（不破上限）；这是「簇内调仓换优」而非加杠杆——若新机会并不比被减标的好，纯属来回折腾，直接判不划来。
- 单只主题仓位 ≤ 组合的 20%，总战术仓 ≤ 60%（约 6W 信封）；
- **不追高**：已翻倍、估值分位>80% 的高位主题不买；
- **手续费前置判断**：短炒（预计<30天）用 C 类（免申购费）；长期拿用 A 类（申购费1折）。

## 四、卖出 / 止盈规则（由 AI 结合信号与账面判断）
> 卖出 / 止盈所得 **全额回流闲置资金**——闲置规模因此增长，正是"让闲置越来越多"目标的核心实现路径；回流后闲置可用于下一轮建仓 / 加仓（受上方「弹药保留纪律」约束）。
1. **强制止损**：单只亏损 **>15%** → 卖出（优先级最高，即便持有<7天也卖，止损重于手续费）。
2. **移动止盈（递增回撤版）**：盈利 **+20% 先卖 1/3** 锁利；**剩余部分从最高点回撤 10% 才卖**（让利润在趋势里继续跑）。趋势继续则回撤阈值随盈利放宽：**+50% 再卖 1/3、回撤 8% 止盈；+80% 回撤 12% 清仓**——避免牛市卖飞，又能反转时保住利润。
   —— 相比纯固定止盈，能减少"卖飞"，又能在趋势反转时保住利润。
- **止盈目标参考（软指标·与递增回撤并行·2026-07-22 学习优化新增）**：偏股/行业主题基金持仓**年化达 8%~15%（行业主题波动大可至 20%）**即开始分批止盈，触达先卖 1/3、不一次性清仓；与「递增回撤止盈」互为双保险。来源：百家号《基金赚钱了什么时候该卖》(2026-05-12) + 叩富网《2026年基金定投止盈止损》(2026-07) + 什么值得买《15%分批止盈法》(2026-05-09)（非官方 ×≥2 佐证）。
3. **破位减仓**：近 10 日从高点回撤 **>10%** → 减仓。
4. **过热减仓**：估值分位 **>80%** → 减仓。
5. **负向催化**：行业利空 → 卖出。
- 卖出所得计入**闲置资金**（负数先抵扣）。
- **手续费前置判断**：非止损的常规卖出，优先等持有 **>7 天**再操作（避开 1.5% 惩罚费）；确需短炒用 C 类。

## 十、可用指令（用户说一句即触发，AI 手动执行）

> 本环境自动化不可靠，故采用「用户发触发词 → AI 手动执行」模式。以下指令无需配置定时任务，用户主动说即可。**九条指令互相独立，可单独触发，不必依赖「更新今天」。**

**指令 1：`更新今天`（全集）**
- 触发：收盘后（16:30 后）用户说"更新今天"（含"更新一下""今天更新"等同义）。
- 执行（四步 = 指令2 + 指令3 + 记账 + 防御）：
  1. **记账（四文档必须全同步，缺一不可）**：任何买入/卖出成交后，必须同时更新以下四份文档，不得只改其中几个：
     - `资金账户操作日志.md`：追加流水 + 更新当前账户快照（闲置/持仓）；
     - `持仓速览表.md`：更新持仓明细、资金与收益、观察池信号；
     - `基金模拟投资方案.md`：更新第五节「当前快照」的闲置资金与持仓（避免方案文档滞后）；
     - `观察池.md`：若持仓变化（新建仓→标「已持」、清仓/止损→状态变更或移除），同步状态列。
     > 教训：2026-07-21 建仓恒生科技后曾漏改方案文档，导致闲置显示 2.0W（实为 0.5W），已修正。**此后每笔交易四文档一起落盘，绝不留单。**
     > **市值核算口径（重要·防错）**：每日「估算市值」= **昨日收盘市值 × (1 + 当日涨跌幅)**（复利滚动，**不是按成本重算**）；「累计盈亏」= 当前市值 − 成本。建仓当日昨市值=成本；历史盈亏须累计进基数，**不得每天从成本清零重算**（否则会抹掉前期亏损/盈利，虚增收益）。
     > **建仓日不算收益（重要·防错）**：建仓日 T 日买入按**当日净值**成交（份额 = 买入金额 ÷ 当日净值，C类免申购费），当日标的涨跌幅已含在买入价内，故建仓日「当日涨跌」对你为 0、市值 = 成本、累计盈亏 = 0；**收益自下一交易日（以建仓净值为新基准）起算**，不得把建仓日指数涨幅计入收益（否则虚增）。
     > **份额制核算（重要·统一口径）**：`份额 = (投入金额 − 申购费) ÷ 建仓净值`（一经买入即固定）；`当日市值 = 份额 × 当日净值`；`累计盈亏 = 份额 × (当日净值 − 建仓净值) = 当日市值 − 成本`。**手续费**：C 类（战术仓）免申购费→份额=成本÷建仓净值；A 类（缓冲仓）按 1 折申购费从投入中扣。**建仓净值（真实买入日净值·已落地）**：各基金真实单位净值经东财官方接口 `api.fund.eastmoney.com/f10/lsjz`（带浏览器 Referer）获取。**建仓净值 = 真实买入当日的单位净值**（非 7/21 重置）：机器人/创新药 7/20 建仓→用 7/20 净值，恒生科技 7/21 建仓→用 7/21 净值；缓冲底仓(债/固收+)买入日早于 7/20，需用户给真实买入日再补全份额。份额=成本÷建仓净值（A类=(成本−申购费)÷建仓净值）。**累计盈亏 = 自真实买入日起的真实涨跌幅**，不再把 7/21 前浮亏冻结抹平。7/22 起按真实净值滚动、如实记录。市值/盈亏展示至「分」。

## 十一、组合级风控补强（2026-07-22 学习优化新增）
> 来源（均为非官方，≥2 独立来源佐证采纳，符合来源校验铁律）：和讯网《如何控制基金投资的最大回撤》(2025-11-22)、《基金投资组合的最大回撤该如何控制》(2025-08-03)；叩富网《买了多只行业基金仍亏损,相关性是什么意思?》(2026-03-27)；什么值得买《三周期共振交易法 月线定方向周线找节奏日线抓买点》(2026-06-05)；新华网《新手避坑之仓位管理》(2026-06-15)；和讯网《分批建仓的仓位管理技巧》(2026-07-01)。
> 与既有「单只 15% 止损 + 移动止盈 + 弹药保留 + 提前卖出释放」构成三层防线：**单只 → 组合 → 相关性**。

1. **组合最大回撤止损（补单只止损盲区）**：单只 15% 止损只防个别爆雷，多只同跌时总盘仍可能大跌。设组合级阈值：组合总市值自**阶段高点回撤 >8%** → 触发降仓（先减最弱 / 相关性最高的敞口，腾闲置）；**>12%** → 大幅降至货币基金持币，不硬扛。
2. **相关性控制（防"伪分散"）**：机器人 014881 + 创新药 024329 + 恒生科技 013402(A类) 同属「成长 / 科技 / 港股」风险簇，相关性高，应**合并算同一风险敞口**。规则：单一相关性簇敞口 ≤ 组合的 **45%（约 4.5W）**；已满则**不追加该簇**，优先分散到**低相关**容器（缓冲债 / 固收+ / 黄金避险 / 货币基金）；新建仓先评估与现有持仓的相关性，避免"买了一堆、实则是同一笔赌注"。
3. **多周期共振过滤（月线定方向、周线找节奏、日线抓买点）**：日线"好"信号需大周期确认——①**月线处于下行 / 熊市结构**时，即便日线信号好也只小仓试探或观望（防周期错配被洗出）；②**周线确认向上**才放大仓位；③日线只负责择时买点。杜绝"日线看多、月线看空"的错配亏损。
4. **分批建仓头寸（首笔 ≤ 计划仓位 50%）**：新建仓 / 加仓不一次性打满——首笔 ≤ 该标的**计划总仓位的 50%**，剩余留作回踩补仓弹药；可用「等额分批」（每跌固定幅度补同等金额）或「金字塔」（越跌买越多）。与「弹药保留纪律」叠加：首笔后闲置仍 ≥ 弹药底线 5,000。
5. **时间止损（给判断设期限）**：持仓 **20~30 个交易日**仍不涨、未触发任何买卖信号、且明显跑输同类 / 大盘 → 视为判断失误，主动退出换更好的标的（卖出所得全额回闲置），不无限期"等回本"。与「提前卖出释放弹药」共用「划得来」判断。
6. **极端估值预警（区分行业·2026-07-23 复盘优化）**：估值高低不能一刀切，按行业属性区别判断：
   - **传统/周期行业**（银行/地产/周期制造等）：PE > 100 倍预警，高概率站岗——这些行业 PE 天然低，高 PE = 严重泡沫。
   - **高成长行业**（半导体/AI/创新药等）：绝对 PE 天然偏高（半导体设备材料合理 60-90 倍、AI 芯片 70-110 倍），改用 **PEG（市盈率÷净利润增速）** 判断——PEG > 2 预警（不建仓），PEG < 1.5 可考虑；配合估值分位 < 80% 作为辅助参考。
