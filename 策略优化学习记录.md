## 一、联网学到的新知识（来源 + 要点）

### 1. 2026 基金赎回费新规（已落地）— 最重要
- 来源：《基金赎回费新规2026年起执行》(什么值得买, 2026-06-24)；《2026年基金新规落地!短期炒基成本飙升,C类份额不香了?》(中宏网, 2026-01-06)
- 要点：
  - 赎回费改为**阶梯且 A/C 统一**：持有 <7天 ≥1.5%；7~30天 ≥1.0%；30~180天 ≥0.5%；>2年多数免。
  - 这部分费用**全额计入基金财产**（不再归销售渠道）。
  - **旧认知"C类7天免赎回费"已失效**——C类短期（7~30天）也要收 ≥1.0%。C类优势只剩"免申购费+低销售服务费"。
- 影响：直接推翻原方案"7~30天≈0.75%""C类7天免赎"的判断，短炒/网格成本被严重低估。
- 〔2026-07-21 **复核修正·官方来源**〕本条原据非官方（什么值得买/中宏网），结论需修正：**证监会〔2025〕22号正式稿对「个人持有满7日的指数型基金」给予豁免、可另行约定赎回费**。我们持有的机器人014881C/创新药C(均C类)/恒生科技013402A(A类) 均为指数型联接 → **满7天通常免赎回费(0%)**，"C类短期也要≥1.0%"仅适用于非指数C/A类。故战术盘指数C**满7天即可免赎回费正常卖/止盈**，不必等30天（原"等>30天"过保守，已改）。阶梯<7天≥1.5%等仍成立。来源：中国政府网公报〔2025〕22号 + 证监会csrc.gov.cn + 贵州证监局（官方，免佐证）。

### 2. 移动止盈：回撤阈值随盈利递增
- 来源：《基金"止盈"到底怎么设?三种止盈策略》(百度百家号, 2026-07)；《基金盈利20%要不要止盈?》(2026-07-09)
- 要点：盈利越高，越能容忍更大回撤（让利润奔跑）。实战参数：盈利 50% 后回撤 >8% 清仓；盈利 80% 后回撤 >12% 离场。比固定 10% 回撤更能避免"卖飞"。
- 影响：原方案固定"回撤10%止盈"可升级为递增阈值。

### 3. 网格间距必须覆盖手续费
- 来源：《最全的网格交易教程》(CSDN, 2026-05)；《手把手!2026版网格交易实操攻略》(什么值得买, 2026-03)
- 要点：网格间距太小会被手续费吃光；需结合标的波动率设定步长；单档金额控制。
- 影响：原方案"间距≥3%"在新规下不够（短持赎回费≥1%，一来回>2%），应提到 ≥5%。

### 4. 动量轮动的量化参数（让"好/不好"更客观）
- 来源：《动量轮动策略 2026上半年总结》(知乎, 2026-06)；《动量轮动策略详解》(myinvestpilot, 2026-07)；《我用一套"四道纪律"ETF轮动策略》(知乎, 2026-07)
- 要点：
  - 用 **20 日涨幅排名**做轮动，动量 > 0 才入选；
  - **跌破 MA60 且动量转负**不入选；
  - 回撤水位 **-8%** 预警、逼近止损线；
  - "该持币"也是信号：所有标的动量均负/破位 → 100% 持币观望，不硬做。

### 5. 底仓 + 定投 + 网格 + 动量协同
- 来源：《量化复合策略设计 ETF底仓定投网格交易动量止盈协同》(CSDN, 2025-09)
- 要点：核心持仓用动量持有（吃趋势），闲置资金在区间做网格（吃波动），二者不冲突。

### 6. 不追鱼尾 / 控制交易频率（原方案已有，资料佐证）
- 来源：原方案第八节已整理；本轮检索再次印证"A股板块月均轮动 5.2 次""年交易>12次手续费显著侵蚀"。

## 四、2026-07-22 学习优化（指令4 · 组合级风控补强）

### 1. 联网学到、且手册原本没有的新知识（来源 + 要点）
- **组合最大回撤止损**：单只 15% 止损只防个别爆雷，多只同跌时总盘仍可能大跌；应设组合级回撤阈值（>8% 降仓、>12% 转货币）。来源：和讯网《如何控制基金投资的最大回撤》(2025-11-22) + 和讯网《基金投资组合的最大回撤该如何控制》(2025-08-03)（非官方 ×2 独立，佐证采纳）。
- **相关性 / 伪分散**：持有多只同风格基金 ≠ 分散，若相关性高实为同一笔赌注；应衡量相关性、控制单一风险簇敞口。来源：叩富网《买了多只行业基金仍亏损,相关性是什么意思?》(2026-03-27)（非官方，佐证采纳）。
- **多周期共振（月/周/日）**：月线定方向、周线找节奏、日线抓买点；月线下行时即便日线好也只小仓，防周期错配。来源：什么值得买《三周期共振交易法》(2026-06-05) + 多个财经号（非官方，佐证采纳）。
- **分批建仓头寸（首笔 ≤50%）**：首笔不一次性打满，金字塔 / 等额分批，留弹药应对波动。来源：新华网《新手避坑之仓位管理》(2026-06-15) + 和讯网《分批建仓的仓位管理技巧》(2026-07-01)（非官方 ×2，佐证采纳）。
- **时间止损**：持仓设期限（20~30 交易日不涨 / 跑输即退出换标的）。来源：中金《详解止损机制》(2024-07-09) 及通用风控常识（非官方，佐证采纳）。
